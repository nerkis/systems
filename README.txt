Using Student

	To use Student, simply use the make command while inside the student 
	directory. Since the program will be named "student" use ./student to 
	run the program. Navigate by typing in information and pressing enter.

Features

	- Allocates space for an array of n students
	- Requests information from the user
	- Enters user-provided information into the array
	- Prints information
	- Tallies the number of each grade recorded
	- Releases the array space when finished
